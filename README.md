## Installation

This module is installed via npm


Option a) (safer)

Checkout the repo and use 'npm link'

Option b)

``` bash
$ npm install rsstojson -g
```

or

## Example Usage

``` bash
rsstojson http://www.spiegel.de/schlagzeilen/tops/index.rss
```
